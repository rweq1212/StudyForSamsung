공지
====

### **1. 스터디 시간**
```
* 월 / 목 (13:00~16:00)
* 13:00~14:00(과제 Review)
* 14:00~15:00(문제 Solving)
* 15:00~16:00(문제 Review)
```

### **2. 장소**
```
>* 신촌 모아스터디
```


### **3. 과제**
```
* 주 4문제(월 : 2문제 할당, 목 : 2문제 할당)
```


### **4. 회비 및 벌금**
```
* 월회비 : 24000원(모아스터디 1인 1시간 : 1000원 X 3시간 X 2회 X 4주
* 벌금 : 결석(5000원) / 지각(3000원)
```

* Solving Problem폴더에 있는 각자 이름 폴더에 저장하면서 사용하시면 됩니다.
